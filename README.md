# prime-count-lab
一、实验目的

掌握Linux环境下多线程程序的编写方法（pthread库使用）

理解多线程对程序性能的影响

学会分析CPU核心数与线程数的关系

二、实验任务

从以下问题中任选一个，分别实现单线程和多线程版本，对比分析效率差异：

选题1：素数统计

任务：统计1到N之间的素数个数

N建议取值：10^7, 5×10^7, 10^8

选题2：立方和计算

任务：计算 1³ + 2³ + 3³ + ... + n³

N建议取值：10^7, 5×10^7, 10^8

选题3：自选题

例如：

- 排序算法（大数据量排序）

- 矩阵乘法

- 或其他你感兴趣的计算密集型问题
